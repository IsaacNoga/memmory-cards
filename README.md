# Memory Cards Game

A simple memory card game built with Astro, React, and TailwindCSS.

## Tech Stack

- Astro
- React
- TailwindCSS
- TypeScript
- Vite

## 🚀 Project Structure

Inside of your Astro project, you'll see the following folders and files:

```text
/
├── public/
│   └── files.svg
│   └── files.mp3
├── src
│   ├── assets
│   │   └── files.svg
│   ├── components

│   ├── layouts
│   │   └── Layout.astro
```

## 🧞 Commands

All commands are run from the root of the project, from a terminal:

| Command                | Action                                           |
| :----------------------| :------------------------------------------------|
| `pnpm install`         | Installs dependencies                            |
| `pnpm dev`             | Starts local dev server at `localhost:4321`      |
| `pnpm build`           | Build your production site to `./dist/`          |
| `pnpm preview`         | Preview your build locally, before deploying     |
| `pnpm astro ...`       | Run CLI commands like `astro add`, `astro check` |
| `pnpm astro -- --help` | Get help using the Astro CLI                     |

## Justification of the project

This project was created using due to Astro's capability to work with multiple JavaScript 
frameworks, good integration with vercel deployment and simple
configuration of the project.

The selected JavaScript framework was React for its state management and light implementation
with astro

As for the css framework, tailwind currently has good support with documentation and ai tools.
